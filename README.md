AlleywayMonoGame

Ein kleines Alleyway-ähnliches Spiel mit MonoGame (DesktopGL).

Voraussetzungen:
- .NET 6.0 SDK oder neuer
- MonoGame 3.8 DesktopGL (installiere mit `dotnet new --install MonoGame.Templates.CSharp`) oder über NuGet

Bauen & Ausführen (PowerShell):
1. dotnet restore
2. dotnet build
3. dotnet run --project AlleywayMonoGame

Steuerung:
- Pfeiltasten links/rechts oder A/D: Paddle bewegen
- Leertaste: Spiel starten / Ball schießen

Dateien:
- Program.cs
- AlleywayMonoGame.csproj
- Game1.cs
- Content/ (leer, verwendet keine externen Assets)
